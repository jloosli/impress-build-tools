Automation

Grunt

  • Concatenate & minimize scripts
  • Run unit tests
  • Minimize images
  • Remove debugging
  • Compile less
  • Auto refresh

Yeoman

Combines bower & grunt. It provides structure for new projects. It can also provide generators for adding new items (think of cake build or laravel artisan)
Put Yeoman in practice npm install -g grunt-cli yo bower
npm install -g generator-cg-angular

cd WidgetCounterApp
